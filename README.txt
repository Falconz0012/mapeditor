This project is a simple Map Editor made with the simpleGFX library for Java.
Features include:
Paintable, erasable squares:Key SPACE
Color options including: Green, Yellow, Red, Pink, Orange, Cyan, Black: key T(change colors)
Clear board feature with repaintability:Key C (clear) Key R (repaint);
Save squares for next session, load from previous one : Key P(save) Key L(load)
